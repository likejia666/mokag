1.1 项目名称
------------
自动化装备招聘系统

1.2 项目定位
------------
为自动化装备领域的求职者与招聘方搭建一个高效、安全、垂直化的招聘与沟通平台

1.3 核心特性
------------
- 垂直化工种管理
- 三级技能分类（大工/中工/小工）
- 实时通讯系统
- 企业认证体系
- 智能举报监管
- 拟态风格UI设计
- 人才库管理

1.4 用户角色
------------
- 求职者：浏览职位、投递简历、实时沟通
- 发布者（雇主/劳务）：发布职位、管理报名、企业认证
- 管理员：平台运营、审核监管、数据统计
