## 快照格式说明

每个快照条目包含：

| 字段 | 说明 |
|------|------|
| 快照ID | 格式：`{任务编号}-S{步骤序号}`，如 M1-006-S3 |
| 时间 | 完成该步骤的时间 |
| 步骤 | 步骤名称（与项目进度跟踪文档步骤清单对应） |
| 产出文件 | 本步骤产出的代码文件路径 |
| 关键内容摘要 | 代码核心逻辑简述（1-2句） |
| 遗留问题 | 本步骤未解决的临时方案/待优化项 |
| 技术债务ID | 如有临时方案，对应项目进度跟踪文档§9的债务ID |
